# newBot

A Node.js version of the Instagram bot, migrated from PHP.

## Structure
- index.js: Main entry point
- bot.js: Telegram bot logic
- config.json: Configuration
- accounts.json: Instagram accounts
- explore.js, followers.js, following.js, hashtag.js, post.js, search.js, start.js: Instagram and bot features
